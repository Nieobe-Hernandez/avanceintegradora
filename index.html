SUGEL

Sistema Universitario Gestor de Examenes en Linea

Sugel es un sistema que permite a los docentes y alumnos realizar evaluaciones de manera eficaz. Permite a los docentes realizar exámenes diferentes para cada alumno reduciendo el tiempo de elaboración para cada examen y obtener información sobre ellos de forma inmediata generando un resumen de la evaluación aplicada.
